Description

GitStarted is the perfect place for exploring Open Source Projects on GitHub. Get project recommendations based on the users past Open Source contributions just by signing in with the users GitHub credentials.


Based on the languages the user has used, the number of stars and forks that the player has on his/her owned/contributed repositories, we show you a list of projects that meets the players current level of expertise in Open Source development.


This project was built as a part of Mumbai Hackathon 2017

Features
  1. A bar graph illustarting the different programming languages used by you in your GitHub based Open Source projects.
  2. A search box to filter out projects from our recommended list based on any keyword.
  3. A graph illustrating the various languages that has been used in the selected project.
  4. Render README.MD if any in the selected project.
  5. Display and render issues if any in the selected project.




Role & Contribution

I was one of the 3 developers who developed this project. I contributed the following features:

  1. Retrieve user informations and calculate recommendation factor.
  2. Retrieve recommended projects using various GitHub wrappers.
  3. Search from recommended projects based on a keyword.

Learning Experience
