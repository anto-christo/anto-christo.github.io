Description

A fun space-themed game built using JavaScript for Teknack 2017. The player has to control his spaceship and avoid hitting the incoming obstacles while at the same time collecting fuel boxes in order to stay afloat in the game.


Features
  • 1. Immersing music and sound effects.
  • 2. Endless gameplay.
  • 3. Online leaderboard.



Role & Contribution

I was one of the 3 developers who developed this project. I was responsible for coding the entire gameplay of the game.

Learning Experience

Space Rush marked many 'firsts' for me. This was the first game I've ever developed. This was the first time I worked with Phaser.js, Node.js-Express.js-MongoDB. This was the first time I was exposed to a version control tool like Git. All the above technologies were taught to me by my seniors at the time.


I'll ever be grateful to them because Space Rush marked the beginning of my journey with game development and GitHub. I went on to make 2 more games after this and did multiple projects on Node.js and MongoDB. Also, it created an interest in me towards contributing to Open Source.
