Description

Programming languages like JavaScript and Python are popular because of their developer-friendly nature. They have hundreds of inbuilt libraries consisting of ready-made functions that make development very easy. C, unlike JavaScript and other similar languages lack such functions. Ease C was an attempt to develop libraries in C which would consist of many user-friendly functions that would make coding with C a lot more fun and easier. Another motive was to provide a source for interested people trying to learn C, with example codes for performing various complex operations in C.

This project was selected as the 2nd Runner Up of Mumbai Hackathon 2017.

Features
  • 1. Three libraries (vector.h | bin.h | array.h).
  • 2. Ninty functions.
  • 3. Documentation available.


Role & Contribution

I was one of the 5 developers who developed this project. Out of the 90 functions, I contributed in developing the entire vector.h library consisting of 22 functions. The vector.h library consists of functions which simplify working with Linked Lists which is otherwise quite difficult in C.

Learning Experience

Working with Linked Lists in C is not everyone's cup of tea. I myself had realized this by seeing my classmates struggle with this data structure during our Second Year of Engieering in Data Structure laboratories.


I was quite confident with Linked Lists and found it very interesting and hence took up the role of developing the vector.h library. This gave me an opportunity to implement whatever I had learned in Data Structures course and made me confident to try out more complex data structures at that time.
