Back View on GitHub WebRTC Video Calling
Description

A peer-to-peer web-based video calling application built entirely using WebRTC and Node.js. WebRTC (Web Real Time Communication) allows browsers to communicate directly with each other without the overhead of using a server. This project demonstrates how WebRTC can be used to build peer-to-peer video calling application using Socket.io as a signalling mechanism and without using a dedicated media server.

Features
  • 1. Option to enable/disable audio.
  • 2. Option to enable/disable video.
  • 3. Toggle fullscreen.




Role & Contribution

I was the sole developer of the project. The entire project with all the mentioned features was developed by me.

Learning Experience

The project was very challenging at the beginning as WebRTC was totally a new concept for me. I had to go through a lot of blogs and videos to understand the working of WebRTC. Once I understood WebRTC, the next challenge was implementing it.


I first went through a lot of tutorials available online but these were either too outdated or too difficult to grasp through. Finally, I began searching on GitHub for example codes and this project came as a blessing. It was well coded and easy to understand and gave me a headstart.


Though not an expert, now I know what WebRTC is and how to implement it.
