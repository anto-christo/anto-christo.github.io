What I'm good at ?

I'm good at driving cars. Sounds odd? Were you expecting me to say I'm good at programming and love to code in so and so langauge? Well, that's how I view programming as, it's like driving a car.


When you say you know to drive a car, it means you can drive any car irrespective of the brand, model or type . Well at first you need some time to get accustomed to a particular car but once you get the hold, you are good to go.


Some cars have automatic transmission and features that reduce the work of the driver, while some have manual transmission and features that require extra work from the driver. In both cases, you just need to read the manual and find out what type of controls the car offers.


It's the same case with programming languages. When you say you know to code, you can code in any language with some initial practice. Some languages are like cars with automatic transmission (Love you Javascript !!) that reduce your work while some are like cars with manual transmission (Hate you Java !!). Again, in both cases you just need to read the manual (documentation) for you to get control over it.
